## Introduction
Portfolio website, the following technologies used Spring boot, Spring Security, Hibernate, PostgreSQL, Docker, Liquibase, Maven. 

## Prerequisites
The following technologies required:
  * Java 8 (OpenJDK)
  * Postgres 11.8+
